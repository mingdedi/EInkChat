# E-Ink LLM Chat

一个为 Android 墨水屏阅读器 + Via 浏览器优化的、零依赖的纯前端 LLM 对话工具。

- **单文件部署**：仅一个 `index.html`，内联所有 CSS 与 JS，无外部资源
- **零构建、零框架**：原生 HTML/CSS/JS，开箱即用
- **墨水屏友好**：纯黑白配色、禁用一切动画与过渡、字号 ≥16px、直角边框
- **本地优先**：API 密钥与配置仅保存在浏览器 `localStorage` 中

> 当前版本：**配置导入/导出（Step 3）**。可在「设置」中维护多个模型，每个模型独立配置显示名称、模型 ID、baseURL 与 API Key；顶部下拉栏可在对话过程中随时切换当前模型。支持将全部配置导出为 JSON 文件备份，或从 JSON 文件导入恢复。

## 使用方法

### 在墨水屏阅读器上

1. 将 `index.html` 拷贝到设备本地存储（例如 `/sdcard/EInkChat/index.html`）。
2. 用 Via 浏览器打开 `file:///sdcard/EInkChat/index.html`。
3. 点击右上角「设置」，点击「新增模型」并填写：
   - **显示名称**：在顶部模型切换栏中展示的名称，例如 `GPT-4o mini`
   - **模型 ID**：请求时使用的 `model` 字段值，例如 `gpt-4o-mini`、`gpt-3.5-turbo`
   - **API 地址（baseURL）**：兼容 OpenAI 协议的 Chat Completions 接口根地址，如 `https://api.openai.com/v1`
   - **API 密钥**：你的密钥（`sk-...`）
4. 可重复「新增模型」配置多套服务商；顶部下拉栏可在对话中随时切换当前使用的模型，已选中的模型卡片左侧会有加粗边框并标注「当前」。
5. 关闭设置面板，在底部输入框输入消息，点击「发送」。
6. 在 Android 软键盘下，回车键默认换行；使用 `Ctrl + Enter` 发送（外接键盘时）。

### 配置备份与迁移

- **导出**：在「设置」中点击「导出配置」，页面内会显示配置 JSON 文本，点击「复制全部」或手动全选复制，粘贴到任意文本编辑器中保存为 `.json` 文件即可。（因墨水屏 WebView 下载兼容性限制，不使用浏览器下载功能）
- **导入**：在「设置」中点击「导入配置」，选择之前导出的 JSON 文件即可恢复全部配置。导入会覆盖当前设置，操作前会弹出确认提示。

### 在桌面浏览器调试

直接双击 `index.html`，或用 Chrome DevTools 的 **Device Mode** 模拟 300×400 分辨率验证布局。

## 安全声明（重要）

- 本项目是**纯前端本地工具**，不包含任何后端服务器。
- 你填写的 API 密钥、地址、模型名称等配置**仅保存在当前设备浏览器的 `localStorage` 中**。
- 应用**只会**向你配置的 API 地址发送请求；除此之外不会上传任何数据。
- 清除浏览器缓存、卸载浏览器、或更换设备**会丢失全部配置**。请使用「设置」中的「导出配置」功能定期备份，在新设备上通过「导入配置」恢复。
- 请勿在公共/共享设备上保存密钥。请勿将填写过密钥的 `index.html` 文件直接提交到任何代码仓库或公开分享。
- 本仓库代码**不包含任何密钥**；请使用 `.gitignore` 中列出的命名规则保存你的本地配置文件。

## 开源贡献指南

1. 提交前请运行 `gitleaks detect`（或同类工具）确认无密钥泄露。
2. 不要引入任何前端框架、构建工具或外部依赖。
3. CSS 不使用 CSS Variables、动画、过渡、Flex/Grid 重排。
4. 所有用户可见文本使用中文，代码注释使用英文。
5. 请保持 `index.html` 单文件形态，所有改动须兼容 Android WebView。

欢迎通过 Issue 与 Pull Request 反馈墨水屏机型适配问题。
